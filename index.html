KEXP Player Open Source

KEXP is a radio station in Seattle which I've been listening for few years now. This is a small (unfinished) project that uses KEXP and last.fm APIs to get information about the song being played and allows automatic scrobbling to last.fm and marking songs as "loved". It also uses datatables to filter through the songs played so it's easy to find something you've listened to days before.
